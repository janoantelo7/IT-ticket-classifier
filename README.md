# IT Ticket Classifier
A solution to categorize IT support requests.

## Data Source

This project utilizes the **IT Support Ticket Data** dataset, provided by Parth Patil on Kaggle. 

* **Author:** Parth Patil
* **Dataset:** [IT Support Ticket Data](https://www.kaggle.com/datasets/parthpatil256/it-support-ticket-data)
