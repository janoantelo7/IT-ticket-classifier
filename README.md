# IT-ticket-classifier
A multilingual NLP solution to categorize customer support requests across 8+ languages.

# Data Source

This project utilizes the **Multilingual Customer Support Tickets** dataset, provided by Tobias Bueck on Kaggle. 

* **Author:** Tobias Bueck
* **Dataset:** [Multilingual Customer Support Tickets](https://www.kaggle.com/datasets/tobiasbueck/multilingual-customer-support-tickets)
* **Year:** 2023

If you plan to use or reproduce this work, please ensure you credit the original author of the dataset.
